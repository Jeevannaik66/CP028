# Medical Store Management

This is a simple command-line based medical store management system written in C. It allows users to perform various operations such as adding, displaying, modifying, and deleting medicines, suppliers, and customers.

## Table of Contents

- [Introduction](#introduction)
- [Features](#features)
- [Usage](#usage)


## Introduction

The Medical Store Management system provides functionalities to manage medicines, suppliers, and customers for a medical store. It helps in maintaining inventory and customer records efficiently.

## Features

### 1. Medicine Management
- **Add Medicine:** Allows users to add new medicines with details such as ID, name, price, and quantity.
- **Display Medicines:** Displays a list of all available medicines with their details.
- **Modify Medicine:** Enables users to update the details of existing medicines, including name, price, and quantity.
- **Delete Medicine:** Allows users to remove a medicine from the inventory.

### 2. Supplier Management
- **Add Supplier:** Allows users to add new suppliers with details such as ID, name, and contact information.
- **Display Suppliers:** Displays a list of all registered suppliers with their details.
- **Modify Supplier:** Enables users to update the details of existing suppliers, including name and contact information.
- **Delete Supplier:** Allows users to remove a supplier from the database.

### 3. Customer Management
- **Add Customer:** Allows users to add new customers with details such as ID, name, and address.
- **Display Customers:** Displays a list of all registered customers with their details.
- **Modify Customer:** Enables users to update the details of existing customers, including name and address.
- **Delete Customer:** Allows users to remove a customer from the database.

## Usage

To use this program, compile the source code using a C compiler (such as GCC) and run the executable file. Follow the on-screen menu to perform various operations such as adding, displaying, modifying, and deleting records.
